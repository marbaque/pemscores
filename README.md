## Features

### Customizer Options
Extended options available from the Customizer:

1. Header image on front page
2. Header background color
3. Custom logo
4. Excerpts or full content in index and archive pages

### Featured Images
Posts and pages can have Featured Images.

### Widgetized Areas
1. Sidebar
2. Footer widgets
3. Page sidebar
